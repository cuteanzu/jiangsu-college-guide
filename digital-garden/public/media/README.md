## 总体规范

| 属性 | 要求 |
|------|------|
| 总数 | 24 张（A 组 6 + B 组 10 + C 组 6 + D 组 6） |
| 横图比例 | 16:9 |
| 竖图比例 | 3:4（仅 D 组） |
| 最小分辨率 | 2560×1440 px（横）/ 1920×2560 px（竖） |
| 格式 | JPG |
| 色调 | 偏暗优先 |
| 各组关系 | **互不重复**——氛围用城市风景、发散用校园建筑、展示用校园生活、摄影展示用艺术化特写 |

## 替换步骤

1. 准备四组共 24 张照片

3. 如需修改文件名或坐标，编辑 `src/pages/home/data.ts`
4. 刷新浏览器（Cmd+Shift+R）
